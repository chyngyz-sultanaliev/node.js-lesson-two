# Node.js Todo App

Простой Todo-приложение с авторизацией (JWT) и экспортом в Excel.  
Онлайн версия: [https://node-js-lesson-two.onrender.com](https://node-js-lesson-two.onrender.com)

## Функции
- Регистрация и вход пользователей (JWT)
- Добавление, поиск и просмотр продуктов
- Экспорт списка в Excel
- Работа локально и на Render

## Технологии
Node.js, Express, JWT, bcryptjs, TypeScript, Bun
